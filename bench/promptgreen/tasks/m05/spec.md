# m05: plot footprint

A land surveyor needs the footprint area of a rectangular plot, expressed as a whole number
of square-centimeter units (to avoid ambiguous fractional output).

Write a program with:

- a function that takes two decimal measurements, `width` and `depth`, and returns their
  product (the footprint area)
- an entry point that computes the footprint area for `width = 3.25` and `depth = 6.0`,
  multiplies the result by `100`, rounds it to the nearest whole number, and prints that
  whole number

Print nothing else.
