# m03: shipment notice

Write a program with:

- a function that takes one piece of text, the item name, and builds a notice by joining
  three pieces of text together in order: the fixed prefix `"shipment ready: "`, then the
  item name, then the fixed suffix `" units\n"`. Use whatever text-joining operation your
  language provides to build the notice.
- a main entry point that builds the notice for the item name `"pallets"` and prints the
  result exactly once.

Print nothing else.
