# m07: rectangle area from a composite value

Write a program with:

- a composite value type holding two fields: `width` (a floating-point number) and `height` (a floating-point number)
- a function that takes one such value and returns the product of its two fields (its area), as a floating-point number
- an entry point that builds one such value with `width` set to `6.5` and `height` set to `7.0`, computes its area (a value exactly halfway between two whole numbers), rounds the result to the nearest whole number, and prints that whole number with no other output

Print nothing else beyond that single number.
