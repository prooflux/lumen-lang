# m10: raw memory ledger

Write a program with a main routine that:

- writes four single-byte values, `5`, `250`, `255`, and `1`, into four consecutive raw
  memory slots starting at address `600000` (so the four addresses are `600000`,
  `600001`, `600002`, `600003`)
- reads all four bytes back from those same addresses and adds them together
- prints the total

Print nothing else.
